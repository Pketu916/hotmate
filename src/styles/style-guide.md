# HotMate Style Guide

## Design Philosophy

HotMate follows a warm, earthy design inspired by authentic Indian cuisine and traditional hospitality. The design emphasizes simplicity, elegance, and a connection to nature.

## Color Palette

### Primary Colors

- **Primary**: `#2C1810` - Deep brown, used for text and primary actions
- **Primary Light**: `#3D2418` - Lighter shade for hover states
- **Primary Dark**: `#1A0F09` - Darker shade for emphasis

### Secondary Colors

- **Secondary**: `#D4A574` - Warm gold, used for accents and CTAs
- **Secondary Light**: `#E8C9A0` - Lighter gold for subtle highlights
- **Secondary Dark**: `#B8935A` - Darker gold for hover states

### Neutral Colors

- **White**: `#FFFFFF` - Background and text on dark
- **Off White**: `#FAF8F5` - Light background
- **Light Gray**: `#F5F3F0` - Subtle backgrounds
- **Gray**: `#8B8B8B` - Secondary text
- **Dark Gray**: `#4A4A4A` - Muted text
- **Black**: `#000000` - Strong contrast

## Usage Guidelines

1. **Consistency**: Always use CSS variables from `variables.css`
2. **Responsive**: Use Tailwind's responsive prefixes (sm:, md:, lg:)
3. **Accessibility**: Maintain proper contrast ratios
4. **Performance**: Optimize images and use lazy loading
5. **Semantic HTML**: Use proper HTML5 semantic elements
